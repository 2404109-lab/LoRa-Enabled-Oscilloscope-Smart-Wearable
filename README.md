# LoRa Enabled Oscilloscope Smart Wearable

## Overview

This project presents a compact wearable diagnostic platform capable of real-time signal acquisition, waveform visualization, motion tracking, distance sensing, and long-range wireless communication.

The system integrates an ESP32-CAM microcontroller, SX1278 LoRa module, MPU6050 IMU, VL53L1X ToF sensor, OLED display, and LM358 signal conditioning circuit into a portable wrist-mounted device.
